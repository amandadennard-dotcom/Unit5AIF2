    The Asynchronous Kitchen

    This code uses setTimeout, so some tasks wait before finishing. Drag the tickets into the order you predict they will finish in, then run the routine and watch the timers race.

    Show the code 
    runBtn.addEventListener('click', () => {
    addLog("Order started...");

    setTimeout(() => {
        addLog("Coffee is brewed!");
    }, 5000);

    setTimeout(() => {
        addLog("Eggs are fried!");
    }, 3000);

    setTimeout(() => {
        addLog("Toast popped!");
    }, 2000);

    addLog("Pouring OJ and placing ticket on counter.");
});
